# 🚀 FastAPI API Monitoring with Prometheus and Grafana

This project monitors a FastAPI app using Prometheus and Grafana, and sends alerts via Alertmanager.

## 🔧 Tech Stack

- Python
- Prometheus
- Grafana
- Alertmanager
- Docker + Docker Compose
